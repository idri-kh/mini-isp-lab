
# Mini ISP  Lab

## Overview
This project simulates a service provider network using OSPF and BGP.

## Technologies

- OSPF
- iBGP
- eBGP


## How to Run
1. Import the .unl file into EVE-NG
2. Load router images
3. Apply configs from /configs
